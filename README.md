# Vulnerable Test Cases Repository

## Overview
This repository contains a comprehensive collection of vulnerable test cases for cross-site scripting (XSS) vulnerabilities in both JavaScript and TypeScript applications.

## Structure
- `cross-site-scripting-js/` - JavaScript XSS test cases with simplified, descriptive file names
- `cross-site-scripting-typescript/` - TypeScript XSS test cases with comprehensive coverage

## Purpose
These test cases are designed for:
- Security testing and analysis
- Vulnerability detection tool validation
- Educational purposes for understanding XSS patterns
- Benchmarking security scanning tools

## Usage
Each test case represents a different XSS vulnerability pattern found in real-world applications. The files have been simplified with descriptive names for easier navigation and understanding.

## Contributing
When adding new test cases, please follow the naming convention of using descriptive, simple file names that indicate the vulnerability type or context.
